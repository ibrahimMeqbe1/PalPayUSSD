# PalPay USSD Code Generator - تطبيق Android

تطبيق Android أصلي لتوليد أكواد USSD الموحدة لمحفظة PalPay. يعمل بدون إنترنت ويمكن تثبيته مباشرة على أجهزة الأندرويد.

## المميزات

✨ **توليد سريع للأكواد** - توليد أكواد USSD في ثوانٍ معدودة
🔌 **بدون إنترنت** - يعمل بشكل كامل بدون اتصال إنترنت
📱 **تطبيق أصلي** - تطبيق Android أصلي مع واجهة احترافية
✅ **دقة عالية** - أكواد دقيقة تتبع معايير PalPay الرسمية
🎨 **واجهة حديثة** - تصميم احترافي مستوحى من ألوان PalPay
🌍 **دعم اللغة العربية** - واجهة كاملة باللغة العربية
📋 **نسخ وسهولة المشاركة** - انسخ الكود أو شاركه مع الآخرين

## التثبيت

### الطريقة 1: تحميل ملف APK مباشرة
1. حمّل ملف `PalPayUSSD.apk` من الإصدارات
2. افتح الملف على جهازك
3. اضغط "تثبيت"
4. سيظهر التطبيق على الشاشة الرئيسية

### الطريقة 2: البناء من المصدر

#### المتطلبات
- Node.js 18+
- npm أو yarn
- Expo CLI

#### خطوات البناء
```bash
# تثبيت المكتبات
npm install

# تشغيل التطبيق على محاكي Android
npm run android

# أو تشغيل على جهازك الفعلي
npm start
```

## كيفية الاستخدام

### خطوة 1: أدخل رقم المستقبل
- أدخل رقم الهاتف الكامل (7-15 رقم)
- مثال: `201001234567`

### خطوة 2: أدخل المبلغ
- أدخل المبلغ المراد تحويله
- يدعم الأرقام العشرية (مثال: `100.50`)

### خطوة 3: انسخ أو اتصل
- **نسخ الكود**: انسخ الكود واستخدمه حسب الحاجة
- **مشاركة**: شارك الكود مع الآخرين

## صيغة الكود

يتم توليد الأكواد بالصيغة التالية:
```
*370*1*1*{رقم_المستقبل}*{المبلغ}#
```

مثال:
```
*370*1*1*201001234567*100#
```
