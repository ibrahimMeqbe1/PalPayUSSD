# تعليمات بناء وتثبيت تطبيق PalPay USSD Generator

## الطريقة 1️⃣: التثبيت المباشر (الأسهل)

### الخطوة 1: تحميل ملف APK
1. اذهب إلى [Releases](https://github.com/ibrahimMeqbe1/PalPayUSSD/releases)
2. حمّل أحدث ملف `PalPayUSSD.apk`

### الخطوة 2: تثبيت التطبيق
1. انقل الملف إلى هاتفك (عبر USB أو البريد الإلكتروني)
2. افتح مدير الملفات على هاتفك
3. ابحث عن ملف `PalPayUSSD.apk`
4. اضغط عليه لبدء التثبيت
5. اضغط "تثبيت"
6. سيظهر التطبيق على الشاشة الرئيسية

---

## الطريقة 2️⃣: البناء من المصدر (للمطورين)

### المتطلبات
- **Node.js 18+** - [تحميل](https://nodejs.org/)
- **npm** أو **yarn** - يأتي مع Node.js
- **Android SDK** - [تحميل](https://developer.android.com/studio)
- **Java Development Kit (JDK)** - [تحميل](https://www.oracle.com/java/technologies/downloads/)

### خطوات البناء

#### 1. استنساخ المستودع
```bash
git clone https://github.com/ibrahimMeqbe1/PalPayUSSD.git
cd PalPayUSSD
```

#### 2. تثبيت المكتبات
```bash
npm install
```

#### 3. تحضير المشروع
```bash
npx expo prebuild --clean
```

#### 4. بناء ملف APK
```bash
cd android
./gradlew assembleRelease
```

#### 5. البحث عن ملف APK
سيكون الملف في:
```
android/app/build/outputs/apk/release/app-release.apk
```

## الطريقة 3️⃣: البناء باستخدام EAS (الأسهل للمطورين)

### المتطلبات
- حساب Expo مجاني - [إنشاء حساب](https://expo.dev)

### خطوات البناء
```bash
# تثبيت EAS CLI
npm install -g eas-cli

# تسجيل الدخول
eas login

# بناء APK
eas build --platform android --local
```

## استكشاف الأخطاء

### المشكلة: "لا يمكن فتح ملف APK"
**الحل**: تأكد من تفعيل "تثبيت من مصادر غير معروفة" في إعدادات الأمان

### المشكلة: "خطأ في التثبيت"
**الحل**: احذف النسخة القديمة من التطبيق أولاً

### المشكلة: "خطأ في البناء: Java not found"
**الحل**: تأكد من تثبيت JDK وإضافته إلى متغيرات البيئة

### المشكلة: "خطأ في البناء: Android SDK not found"
**الحل**: تأكد من تثبيت Android SDK وتعيين `ANDROID_HOME`
